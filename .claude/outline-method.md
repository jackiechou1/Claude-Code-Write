# 古代权谋大纲创作方法论

## # 价值观与内核

古代权谋小说的“魂”是>><#人物的“权欲”和“权谋”#>。以角色的命运沉浮为表象，承载着对权力、人性、道义的深刻反思。

**<<核心价值观>>**
- **聚焦权力斗争**（宫廷、朝堂、江湖等，见父权/君权）
- **探讨人性幽暗**（背叛、嫉妒、贪婪、复仇等）
- **强调智谋博弈**（计策、布局、反间、连环等）
- **追求家国情怀**（忠诚、道义、牺牲）<<可选，但会拔高立意>>
- **设定宏观背景**（乱世、末代、新朝、外敌环伺等）
- **最终回归秩序**（无论过程多乱，结局要稳定，符合主流价值观）

## # 故事四要素

古代权谋故事，**人物**是核，**欲望**是动力，**权谋**是手段，**情节**是骨架。

### ## 一、核心欲望与冲突

**公式**：
**人物A + 核心欲望 + 障碍 = 核心冲突**

### ## 二、角色塑造

**要素**：[欲望] + [性格] + [能力] + [弱点]

**<<角色设计黄金三问>>**
1. **ta最想要什么？**（驱动力）-> [欲望] -> “他/她是一个<#欲望#>的人”
2. **ta会如何做？**（行为模式）-> [性格] + [能力] -> “他/她会用<#手段#>去实现”
3. **ta的软肋是什么？**（冲突点）-> [弱点] -> “但<#弱点#>会成为他/她的阿喀琉斯之踵”

**<<人物示例>>**
- “他是一个**野心勃勃**的皇子，**擅长隐忍和布局**，他渴望皇位，**但他对亲情的贪恋**是他的软肋”
- “她是一个**渴望自由**的宫女，**聪慧机敏**，她想逃离皇宫，**但她的家族**是她无法割舍的枷锁”

### ## 三、叙事结构

**## 试错**
- **开局**：主角遭遇困境，展露野心，初试牛刀，小获成功
**## 试错 (起)** - **开局**：主角遭遇困境，展露野心，初试牛刀，小获成功。（这构成了“起”，即故事的开端） - **发展**：冲突升级，主角遭遇第一个真正的“敌人”或“障碍”。主角运用计谋（初级权谋），解决小麻烦，但也引来更大的注意和危机。

**## 博弈 (承)** - **交锋**：主角与核心对手（反派）正式交锋。双方你来我往，互有胜负。情节开始深入，展示更复杂的权谋手段。 - **布局**：主角开始设下更长远的局。这期间可能会有盟友的加入、关键信息的获取，或是遭遇背叛，情节跌宕起伏。 - **危机**：对手设下致命陷阱，或利用主角的[弱点]进行打击。主角陷入最大危机（“All is Lost”时刻），失去重要的人或物，被逼入绝境。

**## 破局 (转)** - **反转**：主角在绝境中沉思，找到破局点（通常是利用了之前埋下的伏笔、对手的性格弱点，或是“人性”的幽暗面）。 - **决战**：终极权谋对决。这不是兵刃相见，而是计策的“收网”。主角揭露所有真相，或以雷霆手段将对手彻底击败。这是故事的高潮和转折点。

**## 收尾 (合)** - **结局**：主角达成目标（登上高位、复仇成功、或得到想要的“秩序”）。 - **代价**：展示主角为此付出的代价（例如失去纯真、众叛亲离、或变成自己曾经讨厌的人），呼应“探讨人性幽暗”的价值观。 - **新秩序**：建立新的平衡，或展示一个循环的开始。呼应“最终回归秩序”的核心价值观。

## # 权谋设计（核心技巧）

权谋的本质是“信息差”和“人性利用”。

1. **信息为王**：权谋的核心是信息。设计主角如何获取、隐藏、筛选、利用甚至制造“假信息”来误导对手。 2. **人性为饵**：最高级的权谋是利用对手的[弱点]（贪婪、多疑、恐惧、傲慢，甚至是他们的“美德”如仁慈或亲情）。 3. **伏笔与收网**：没有无缘无故的计策。前面随手布下的一颗闲棋（伏笔），必须在“破局”阶段成为致胜一击（收网）。 4. **连环计**：计策必须环环相扣。A计划是为了掩护B计划，B计划的真实目的可能是为了触发C计划。 5. **“借刀”**：主角（特别是前期弱小时）不应亲自出手，而应善于利用第三方力量（其他敌人、规则、舆论）来达到目的。

## # 氛围与风格

1. **文白相济**：语言要有古风古韵，但不能晦涩难懂。用现代的叙事节奏，包装在半文言的语言风格下，营造“高级感”。 2. **张力（Tension）**：多用“暗斗”。宴会上的一个眼神，一句双关语，比直接拔刀更有权谋感。 3. **节奏**：短篇小说（2000-3000字）节奏必须快。每一章都要有信息点、小冲突或反转。拒绝“水”情节，确保每一段文字都在推动“博弈”的进程。 4. **冷峻**：权谋的底色是冰冷和理性的。即使描写最激烈的情感（如复仇），也要用克制的笔触，展现人物在算计下的“不动声色”。
