# ✍️ ClaudeCode 短篇小说提示词工具

## 🌟 项目简介

**ClaudeCode 短篇小说提示词工具** 是一款旨在为内容创作者、作家和爱好者提供**高质量、专业级提示词**的解决方案。

我们的核心价值在于：**专注于短篇小说的创作需求**。我们提供的提示词不仅是简单的故事情节起点，更是深度优化、结构清晰的指令集，确保您能充分发挥 **Claude/ClaudeCode 模型**在短篇叙事、角色塑造和场景描写上的强大能力。

无论是寻找灵感、构建复杂世界观，还是需要对现有草稿进行结构性调整，本工具都能助您一臂之力，**快速生成引人入胜的短篇佳作。**

## 快速上手指南

### 如何使用提示词：

1.  **选择主题：** 确定您想要创作的小说主题或类型。
2.  **获取提示词：** 复制本项目中相应模块的提示词。
3.  **粘贴至 Claude：** 将提示词完整粘贴到 Claude/ClaudeCode 的输入框中。
4.  **模型输出：** Claude 将根据高度优化的提示词开始生成您的短篇小说草稿。
