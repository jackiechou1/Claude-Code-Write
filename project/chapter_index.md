# 章节目录

- **第1章**：冷宫夜会
    - 失宠十年的七皇子萧珩深夜迎来神秘宫女苏瑾，两人以密信为契，结成复仇与夺权的暗盟。

- **第2章**：密信风波
    - 萧珩设局让密信落入宁公公手中，引发太子与贤妃的猜忌裂痕，暗流涌动初现端倪。

- **第3章**：母子生隙
    - 伪造的密奏挑拨贤妃与太子关系，苏瑾身份险遭暴露，萧珩陷入被动绝境。

- **第4章**：天罗地网
    - 萧珩向天子自揭真相，联合策反势力与关键证据，宁公公倒戈，太子与贤妃众叛亲离。

- **第5章**：暗御孤行
    - 太子贤妃伏诛，苏家沉冤昭雪，萧珩掌权却迷失本心，苏瑾复仇毕转身离去。

[注：根据《大纲》设置5个章节，与大纲的5个阶段一一对应。每章概要写清楚核心看点，精简在30-50字]
